Cloud Web Infrastructure: CI/CD, Monitoring, and Security

The main goal of this project is to automate application deployment, ensure real-time monitoring, and enhance security using modern cloud technologies.

  • CI/CD Pipeline (Argo CD): Automates application deployment whenever there is a new code update in GitHub.
  • Kubernetes (Minikube): Manages containerized applications efficiently.
  • Helm: Simplifies Kubernetes configurations and deployment.
  • Prometheus & Grafana: Provides real-time monitoring to track system performance.
  • Trivy & SonarQube: Ensures security and code quality before deployment.
